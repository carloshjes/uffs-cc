Níveis de colesterol (mg/dl)

n = 78

INTUIÇÃO

A média usa todos os valores; a mediana localiza a posição central; a moda aponta a classe com maior frequência.

MÉDIA

Σ(Xi · Fi) = 17.350 e n = 78

x̄ = Σ(Xi · Fi) / n = 17.350 / 78 = 222,44 mg/dl
MEDIANA

n / 2 = 39; a 39ª observação cai na classe 200 |- 250.

Me = 200 + ((39 - 26) / 35) · 50 = 218,57 mg/dl
MODA

A classe modal é 200 |- 250, pois possui a maior frequência.

Mo = 200 + ((35 - 24) / ((35 - 24) + (35 - 14))) · 50 = 217,19 mg/dl
RESPOSTA FINAL

Média = 222,44 mg/dl; Mediana = 218,57 mg/dl; Moda = 217,19 mg/dl.
